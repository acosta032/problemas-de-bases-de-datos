# Problemas de bases de datos
Repositorio donde están algunos problemas resueltos de bases de datos del libro Problemas de bases de datos de
Luis Grau Fernandez 
Ignacio Lopez Rodriguez.

El repositorio se organiza de la siguiente manera:
- Las carpetas del directorio raiz contienen los ejercicio de cada capitulo del libro
- A su vez, dentro de cada una hay subcarpetas donde están los problemas enumerados
- Dentro de cada carpeta de problema están:
  - Documento HTML con el enunciado y los componentes del ejercicio.
  - Modelos E-R
  - Modelo relacional
  - Scripts
  - Backups
  
  Sugerencias y colaboraciones son bienvenidas.
